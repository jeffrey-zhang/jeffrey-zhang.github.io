关于本站

网站介绍

这是一个基于 Flask 和 GitHub Pages 搭建的个人日记系统，用于记录和分享个人的想法、学习笔记和生活点滴。

主要功能

  • 支持 Markdown 格式写作
  • 文章分类和标签管理
  • 按日期归档
  • 全文搜索
  • 响应式设计，支持移动端访问
  • 通过 Git 进行版本控制

技术栈

  • 后端：Python Flask
  • 前端：HTML、CSS、JavaScript
  • 部署：GitHub Pages
  • 自动化：GitHub Actions
